# Slate & Oak Contracting — Website

A professional, fully responsive website for Slate & Oak Contracting, Mississauga.

## Files
- `index.html` — Main website
- `style.css`  — All styles (dark green + cream theme)
- `script.js`  — Scroll effects, mobile menu, form handling
- `logo.jpg`   — Client logo (add their logo file here)

## Deploy to GitHub Pages
1. Create a new GitHub repo (e.g. `slate-oak-contracting`)
2. Upload all files to the repo root
3. Go to **Settings → Pages → Source → main branch → / (root)**
4. Site goes live at: `https://yourusername.github.io/slate-oak-contracting`

## Custom Domain (optional)
- Buy `slateoakcontracting.ca` on Namecheap (~$12/yr CAD)
- Add a `CNAME` file in the repo with the domain name
- Point DNS to GitHub Pages IPs
